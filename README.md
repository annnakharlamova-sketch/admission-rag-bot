# Admission Assistant

Интеллектуальный помощник приёмной комиссии МТУСИ, реализованный на основе технологии Retrieval-Augmented Generation (RAG).

Система отвечает на вопросы абитуриентов, используя официальные документы университета, данные с сайта приёмной комиссии и локальную языковую модель, запущенную через Ollama.

---

## Основные возможности

- поиск информации по официальным документам МТУСИ с использованием RAG;
- генерация ответов с помощью локальной языковой модели;
- автоматический парсинг сайта приёмной комиссии;
- поддержка документов в форматах PDF, DOCX, XLSX и TXT;
- распознавание текста в сканированных PDF с использованием OCR;
- семантический поиск по векторной базе данных ChromaDB;
- поиск по базе FAQ;
- веб-интерфейс для взаимодействия с пользователем;
- отображение источников информации в каждом ответе.

## Принцип работы

1. Документы МТУСИ индексируются в ChromaDB.
2. Страницы официального сайта автоматически парсятся и сохраняются локально.
3. Для документов строятся векторные представления (embeddings).
4. Пользователь задаёт вопрос.
5. Выполняется семантический поиск наиболее релевантных документов.
6. Из найденных документов формируется контекст.
7. Контекст передаётся языковой модели через Ollama.
8. Пользователь получает ответ со ссылкой на использованные источники.

## OCR

Если PDF не содержит текстового слоя, система автоматически распознаёт текст с помощью Tesseract OCR.

При отсутствии необходимых библиотек OCR отключается, при этом остальные функции системы продолжают работать.

## Установка

Клонировать репозиторий:

```bash
git clone https://github.com/<username>/mtuci-admission-assistant.git
cd mtuci-admission-assistant
```

Установить зависимости:

```bash
pip install -r requirements-windows.txt
```

или

```bash
pip install -r requirements-linux.txt
```

Запустить Ollama:

```bash
ollama serve
```

Загрузить модель:

```bash
ollama pull qwen3.5:9b
```

Запустить приложение:

```bash
python web_app.py
```
